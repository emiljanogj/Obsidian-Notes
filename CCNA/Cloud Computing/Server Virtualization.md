- 2 Types of Hypervisors:
	- Type 1 - run directly on the system hardware(shown in the picture above)
		- Microsoft Hyper-V
		- Red Hat KVM
		- Oracle VM Server
	- Type 2 - run on top of the host OS
		- VMware Workstation
		- Virtualbox
		- QEMU
